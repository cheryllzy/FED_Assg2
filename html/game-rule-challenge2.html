Guess The Artist - Game Rules


Gameplay

The objective of the game is to correctly identify the artist of a given music albums. Players will be given album name and given 3 attempts to guess the correct song title. If the user used up all 3 attempts, he/she will need to press next question to continue the game. Points are awarded for each correct guess, and the player with the highest score at the end of the game wins.


Scoring

Each question will be awarded 5 points
