Guess The Song - Game Rules


Gameplay

The objective of the game is to correctly identify the title of a given musical track. Players will be given part of the lyrics and given 3 attempts to guess the correct song title. If the user used up all 3 attempts, he/she will need to press next question to continue the game. Points are awarded for each correct guess, and the player with the highest score at the end of the game wins.


Scoring

Easy Level >> Each question answered correctly is awarded 5 points each

Intermediate Level >> Each question answered correctly is awarded 10 points each

Hard Level >> Each question answered correctly is awarded 15 points each
